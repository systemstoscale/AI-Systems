# Contributing to AI Systems Template

Thank you for considering contributing! This template helps students build AI systems.

## Prerequisites
- Understanding of AI agents and Claude Code
- Familiarity with course content (Skool)

## Local Setup

```bash
git clone https://github.com/systemstoscale/AI-Systems.git
cd AI-Systems
```

## Development Workflow

### Conventional Commits

**Examples**:
```bash
feat(templates): add sales outreach automation template
fix(context): correct business model canvas template
refactor(instructions): simplify onboarding SOP
docs(readme): update course prerequisites
```

**Scopes**: context, instructions, templates, course, docs

## Code Standards

### Context Files
- Clear structure
- Use placeholders: [YOUR COMPANY], [YOUR PRODUCT]
- Include examples

### Instructions (SOPs)
- Step-by-step numbered steps
- Copy-paste ready code
- Error handling guidance

### Templates
- Working code (test before committing)
- Well-commented
- Easy to customize

## Testing

Test all templates before committing:
- Run scripts to verify they work
- Check n8n workflows import correctly
- Verify examples are clear

## Pull Requests

1. Fill in PR template
2. Identify which course module affected
3. Note if students need to re-download
4. Request review from @max

## Versioning

Use semantic versioning:
- v1.0.0 - Initial release
- v1.1.0 - New templates
- v2.0.0 - Breaking changes
